# 🏠 闺蜜小镇

一个专为14岁女孩设计的多人实时社交游戏平台。和闺蜜一起在小镇里养小人、玩游戏、聊八卦！

## ✨ 功能特色

### 🏠 个人养成
- 拥有自己的2D小屋，布置家具
- 小人四维状态：心情❤️、精力⚡、饱腹🍔、学业📚
- 丰富的换装系统：发型、发色、上衣、下装、颜色自由搭配
- 写日记记录每天的心情

### 🎮 多人游戏
- **🎨 你画我猜**：多人实时画画猜词，比比谁是灵魂画手
- **🎲 冒险棋**：回合制掷骰子，触发各种有趣事件
- **🤝 默契大挑战**：看看你和闺蜜是不是真闺蜜！
- **🎭 真心话大冒险**：闺蜜版，超刺激！

### 👭 社交系统
- 在线好友列表，实时显示在线状态
- 戳一戳功能，叫闺蜜来玩
- 小镇公共聊天室
- 实时多人游戏房间

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 打开浏览器访问
http://localhost:3000
```

### 环境变量

创建 `.env` 文件（可选）：

```
PORT=3000
DATABASE_URL=postgresql://...
```

> 注意：如果不配置 DATABASE_URL，会自动使用内存模式（数据不会持久化）

## 📦 部署到 Render

### 方法一：使用 render.yaml 一键部署

1. 将代码推送到 GitHub
2. 在 Render 中点击 "New > Blueprint"
3. 选择你的 GitHub 仓库
4. 选择 `render.yaml` 文件
5. 点击 "Apply" 等待部署

### 方法二：手动部署

1. **创建 Web Service：
   - Runtime: Node
   - Build Command: `npm install`
   - Start Command: `npm start`

2. **创建 PostgreSQL 数据库：
   - 在 Render 中创建免费版 PostgreSQL
   - 将数据库连接字符串添加到 Web Service 的环境变量 `DATABASE_URL`

## 📱 手机访问

部署完成后，在手机浏览器打开你的 Render 网址即可在手机上玩！
支持触摸操作，完全适配手机屏幕。

## 🛠 技术栈

- **前端**：HTML + CSS + JavaScript（纯原生，无框架）
- **后端**：Node.js + Express
- **实时通信**：Socket.io
- **数据库**：PostgreSQL
- **部署**：Render

## 🎯 游戏玩法

### 你画我猜
- 系统随机选一个人当"画手"
- 画手在手机上画画，其他人猜
- 猜对的人得10分，画手得5分
- 一共3轮，最后总分最高的赢！

### 冒险棋
- 最多4人参与
- 轮流掷骰子，自动前进
- 踩到不同格子触发事件（前进/后退）
- 先到终点的获胜！

### 默契大挑战
- 所有人同时选择答案
- 选得最多的答案就是"默契答案"
- 选到默契答案的人得10分
- 全部一致得20分！

### 真心话大冒险
- 轮流选择真心话或大冒险
- 抽到的人必须执行哦~
